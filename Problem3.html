Question: Write a function: findSumPairs(intArr, sumTotal) that searches an array of integers (intArr, which could have positive and/or negative values) to find pairs that add up to sumTotal.

For example: findSumPairs([-1, 0, 1, 2],3)>[1,2]) findSumPairs([-1, 0, 1,2], 1)>[1, 2], [0, 1])
